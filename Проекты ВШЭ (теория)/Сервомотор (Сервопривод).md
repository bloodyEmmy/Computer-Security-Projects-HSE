Мотор с точным управлением угла поворота. То есть у мотора есть "язычок", угол поворота которого мы задаем. 
Используется для поворота колес автомобиля, башни танка и тд. 
Угол поворота = 0-180 градусов.
Пишем угол => поворачивает язык и оставляет на месте => можно зациклить повороты.
