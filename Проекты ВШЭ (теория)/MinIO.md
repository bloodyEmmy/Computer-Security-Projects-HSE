#flashcards 
***
Облачное хранилище файлов.
Может использоваться вместе с [[Удаленный стенд]]. Стенд может генерировать данные (файлы с видеокамеры, графики, логи экспериментов), их не получается хранить на [[Одноплатный компьютер]] из-за малого кол-ва памяти на борту.
MinIO = централизованное хранилище для всех файлов стенда.
***
Есть преимущество в виде масштабирования (прим: добавлять новые [[Ячейки стенда]]).
Вывод данных из хранилища идет через веб-интерфейс.
